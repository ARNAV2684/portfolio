# AWS Credly badges

Drop the official Credly badge PNGs here (referenced from `data/content.ts`):

- `aws-saa.png` — AWS Solutions Architect – Associate
- `aws-ccp.png` — AWS Cloud Practitioner – Foundational

Download them from your Credly profile (each badge → "Download" → PNG).

Until these land, each cert renders an octagon frame tinted with the credential's
ring color and an "AWS" fallback mark — so the section looks complete either way.
